# AGENTS.md

This repository contains Adobe Illustrator ExtendScript.

## Requirements

- ES3 only
- Do not use `let`/`const`
- Preserve existing functionality
- Use IIFE
- Use `var`
- Compatible with Illustrator 2024–2026
- UI comments are Japanese and English
- Do not remove existing features
- Prefer `doc.selection` over `app.selection`
- Keep `SCRIPT_VERSION` updated
- Localization uses `LABELS` and `getLocalizedText()`
