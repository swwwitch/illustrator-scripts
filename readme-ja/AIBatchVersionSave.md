# フォルダー内のAI／SVGをバージョン指定で一括保存

[![Direct](https://img.shields.io/badge/Direct%20Link-AIBatchVersionSave.jsx-ffcc00.svg)](https://github.com/swwwitch/illustrator-scripts/blob/master/jsx/files/AIBatchVersionSave.jsx)

[![Direct](https://img.shields.io/badge/Back%20to%20home-All%20scripts-cccccc.svg)](https://github.com/swwwitch/illustrator-scripts/blob/master/README.md)

---

### 概要

- フォルダー内の .ai / .svg ファイルを、指定したIllustratorバージョンでまとめて保存するスクリプト。
- .svg はIllustratorで開き、選択したバージョンの .ai として保存される。
- 「上書きモード」と「カスタム」を切り替えて、上書き保存とサフィックス付き別名保存を使い分けられる。
- 処理中はIllustratorの警告ダイアログ（カラープロファイル、フォント置換、上書き確認など）を抑制し、進捗ウィンドウで状況を表示。

### 主な機能

- 保存バージョンをドロップダウンから選択（実行中のIllustratorが対応するバージョンのみ表示。CC以降は v17 に集約）
- 「上書きモード」：対象フォルダー内のファイルを同じベース名の .ai として保存（.ai は上書き、.svg は同名の .ai を生成）。保存先やファイル名の付加設定は無効化される
- 「カスタム」：保存先フォルダーとファイル名の付加設定を自由に指定
- 「フォルダー指定」パネル：［対象...］［保存先...］でフォルダーを選択。「保存先を対象と同じにする」（デフォルトON）で保存先指定を省略可能
- パス表示は「フルパス」「Dropboxパスを短縮」で切り替え（Dropbox短縮ON時はフルパスの切り替えは無効。ツールチップには常にフルパスを表示）
- 「対象」パネル：.ai / .svg のチェックボックス。対象フォルダーに該当ファイルがない拡張子は自動的にディム表示＋OFF
- 「保存設定」パネル：Illustrator標準の［更新済み］付与を使用（処理後に元の環境設定へ戻す）、PDF互換ファイルを作成
- 「ファイル名の調整」パネル：保存バージョン文字列の付加（v17 など。バージョン変更時に自動更新）、任意の文字列の付加、区切り文字（`-` / `_`、デフォルトは `_`）
- ファイル名内の半角／全角スペースやタブを区切り文字に変換し、出力拡張子は常に小文字の .ai に統一
- 上書き事故防止の確認ダイアログ：カスタムモードで .ai 入力があり、対象と保存先が同じでファイル名への付加がない場合／上書きモードで .svg を含む場合
- 進捗ウィンドウに現在のファイル名と件数（n/総数）を表示
- 各項目にツールチップ、日本語／英語の自動切り替え

### 処理の流れ

1. 設定ダイアログでモード、フォルダー、対象拡張子、保存バージョン、ファイル名調整を指定
2. 対象フォルダー内から選択した拡張子のファイルを収集し、必要に応じて上書き確認を表示
3. ［更新済み］付与の環境設定を一時変更し、警告ダイアログを抑制したうえで1ファイルずつ開いて saveAs
4. 進捗ウィンドウを閉じ、環境設定とユーザー操作レベルを元に戻して完了を通知

### 対象外

- 実行中のIllustratorで選択可能な保存バージョンがない場合
- 「対象」パネルで .ai / .svg のいずれもチェックされていない場合
- 対象フォルダー内に該当するファイルが1つもない場合
- .ai / .svg 以外の拡張子のファイル、サブフォルダー内のファイル
