# アートボードサイズ変更

[![Direct](https://img.shields.io/badge/Direct%20Link-AiArtboardScaler.jsx-ffcc00.svg)](https://github.com/swwwitch/illustrator-scripts/blob/master/jsx/artboard/AiArtboardScaler.jsx)

アートボードを少しだけ大きくしたい、あるいは全体を80%に縮めたい、という場面があります。

アートボードツールで引っ張れば矩形は変わりますが、**中身のオブジェクトは置いてけぼり**になります。かといってオブジェクトごと選択して拡大・縮小すると、今度はアートボードとの位置関係が狂ってきます。しかも「何%がちょうどいいか」は、たいてい実際に見てみないと決まりません。

そこで、アートボードとオブジェクトを同じ基準点・同じ倍率でまとめて拡縮しつつ、**ダイアログを開いたまま結果を確かめられる**ようにしました。

## 使い方

1. スクリプトを実行
2. 「対象」でどのアートボードを変更するか選ぶ
3. スケール（%）または幅・高さを入力する
4. 基準点（9つのうちどこを固定するか）を選ぶ
5. ［適用］で確定

ダイアログを開いている間、アートボードの矩形はライブプレビューされます。キャンセルすれば元どおりです。

数値入力欄では `↑` `↓` キーで値を増減できます。`Shift` を併用すると±10で、10の倍数にスナップします。

### 対象

| 選択肢 | 動作 |
| --- | --- |
| 現在のアートボード | スクリプト起動時にアクティブだったアートボードのみ |
| すべてのアートボード | ドキュメント内の全アートボード |
| 指定 | 番号で指定（例: `3, 4` または `3-5`） |

初期状態は「すべてのアートボード」です。「指定」を選んだときだけ入力欄が有効になります。

指定の書式は厳密に検証されます。`3abc` や `1-2-3`、`1,,2` のような空トークンを含む入力、アートボード総数を超える番号はすべて不正扱いです。`5-3` のような逆順は自動的に入れ替えられ、重複した番号は1回だけ処理されます。

### スケールと幅・高さ

スケール（%）・幅・高さの3つの欄は連動しています。どれかを変えると、残りが自動で追従します。

幅・高さの単位は、Illustrator の定規単位（環境設定）に追従します。inch / mm / pt / pica / cm / Q / px / ft / m / yd に対応しています。

**ここが少し分かりにくいところです。** 幅・高さの入力は、**アクティブなアートボードの現在サイズを基準にした%に変換されます**。対象が複数の場合、各アートボードはその同じ倍率で拡縮されます。つまり、幅に `200mm` と入力しても「すべてのアートボードが200mm幅に揃う」のではなく、「アクティブなアートボードが200mmになる倍率が、全部に適用される」という動きです。サイズがばらばらのアートボードを1つの値に統一する機能ではありません。

なお、スケールは常に縦横同率です。幅と高さを別々の倍率にすることはできません。

### 基準点

3×3のウィジェットで、拡縮しても動かない点を9か所から選べます。既定は左上です。

### オブジェクトと一緒に拡大・縮小

初期状態でオンです。アートボード上のオブジェクトを、アートボードと同じ基準点・同じ倍率で拡縮します。線幅も一緒に拡縮されます。

**ロックされたオブジェクト・非表示のオブジェクトは対象外です。** その場合、アートボードの矩形だけが変わります。

オブジェクトの拡縮はプレビューには含まれません。プレビュー中に動くのはアートボードの矩形だけで、オブジェクトは［適用］を押した時点で一度だけ拡縮されます。

### ピクセルグリッドに最適化

初期状態でオフです。オンにすると、新しい幅・高さを整数に丸めます。

基準点の固定と幅・高さの整数化を優先する仕様のため、**基準点も整数に丸められます**。その結果、基準点でない側の端（左上を基準にしたなら右端・下端）は 0.5px 単位になる場合があります。オブジェクトは、丸めたあとの実効倍率で拡縮されるので、アートボードとのズレは生じません。

## 複数アートボードにまたがるオブジェクト

アートボードをまたいで置かれたオブジェクトは、放っておくと両方のアートボードの処理で二重に拡縮されてしまいます。そこで、オブジェクトごとに「どのアートボードのものか」を1つに決めてから変形します。

判定は次の順です。

1. オブジェクトの中心点が入っているアートボード
2. どこにも中心が入っていなければ、重なり面積が最大のアートボード
3. 面積が同じなら、番号が小さいほう

重複の排除には uuid を使い、取得できない場合はオブジェクト参照そのものを比較します。

## 対象

アートボード、およびアートボード上の編集可能なオブジェクト（`selectObjectsOnActiveArtboard()` で選択できるもの）

## 補足

プレビューは意図的にアートボードの矩形だけに限っています。矩形の書き換えは完全に元へ戻せますが、オブジェクトの拡縮は繰り返すたびに誤差が溜まるためです。

そのため、内部では常に**起動時の状態をスナップショットとして保持**し、プレビューのたびにいったんそこへ戻してから計算し直します。［適用］のときも同じで、まず初期状態に戻してから、矩形とオブジェクトを一度だけ適用します。何度スライダーを行き来しても誤差が蓄積しないのはこのためです。

変形や復元に失敗した場合は、そこまでの変形を逆拡縮で巻き戻したうえで中止し、アラートで通知します。確定処理中は座標系を `DOCUMENTCOORDINATESYSTEM` に固定して、`position` / `geometricBounds` を `artboardRect` と同じ座標で扱っています。

基準点ウィジェットの配色は、Illustrator の UI 明度（`uiBrightness`）を見て明暗を切り替えています。
