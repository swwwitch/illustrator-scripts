### 概要

- アートボードを基準にガイドを整理・作成するスクリプト。
- 「ルーラーガイドの変換」「中心ガイド」「エッジガイド」の3系統を1つのダイアログでまとめて設定できる。
- 作成したガイドはすべて「_guide」レイヤーに集約する（無ければ自動作成、ロック／非表示は解除して使用）。
- 設定変更に追従するライブプレビュー付き。

### 主な機能

- **ルーラーガイドを変換**（パネルタイトルに対象件数を表示）
  - アートボードに重なる直線のルーラーガイドを検出し、アートボード基準の直線ガイドに引き直す
  - 「外側に延長」でアートボードの端から外側へ延ばす量を指定（0で端ぴったり）
  - 「すべてのアートボード」ON：ガイドが重なるすべてのアートボードを対象／OFF：最初の1つだけ
  - チェックボックスOFFで変換をスキップ。変換対象が0件のときは自動的に無効化し、その旨を表示
- **中心とエッジにガイドを描画**
  - 「中心（垂直）」「中心（水平）」でアートボード中心のガイドを作成（既定OFF）
  - 「エッジ」マスターをONにすると、上／左／右／下の各辺にガイドを作成（マスターは既定OFF、4辺の個別チェックは既定ON）
  - 「外側へ延長」でアートボードの角から外側へ延ばす量を指定（既定10mm相当）
  - 「すべてのアートボード」ON：全アートボードに描画／OFF：アクティブアートボードのみ
- **プレビュー**（既定ON）：専用レイヤーに色付き線で仮表示し、確定時に本物のガイドへ置換。変換対象の元ガイドは一時的に非表示にする
- 入力値は現在のルーラー単位（rulerType）として扱い、内部でptへ換算
- ↑↓キーで±1増減、Shift併用で±10（10の倍数にスナップ）
- 日本語／英語の自動切り替え、各項目にツールチップ

### 処理の流れ

1. ドキュメント内の全ガイドを収集し、アートボードに重なる直線ガイドを変換対象として事前検出
2. ダイアログで変換・中心・エッジの各設定を指定（変更のたびにプレビューを再描画）
3. OKで、変換対象の元ガイドを削除してアートボード基準のガイドを作成
4. 続いて対象アートボードにエッジガイド・中心ガイドを作成（すべて「_guide」レイヤー）

### 対象外

- ドキュメントが開かれていない場合（警告を表示して終了）
- 非表示のガイド
- 垂直でも水平でもない（斜めの）ガイド
- どのアートボードにも重ならないガイド
- ガイドが1本も無い場合でも、中心・エッジの作成だけは実行可能
