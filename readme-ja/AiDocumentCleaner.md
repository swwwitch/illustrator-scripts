## 概要

ドキュメント内の不要な要素をまとめて削除するクリーンアップツールです。ダイアログで対象を選び、一度に整理できます。

スウォッチ・シンボル・ブラシ・グラフィックスタイルは、各パネルの「未使用をすべて選択 → 削除」アクションを一時生成・再生し、Illustrator 本体の判定で未使用のみを削除します。パス／オブジェクト、グループ／レイヤー、ガイド、アートボードもまとめて対象にできます。

## 主な機能

- **パネル項目**：スウォッチ、シンボル、ブラシ、グラフィックスタイル、段落スタイル、文字スタイル
  - スウォッチ／シンボル／ブラシ／グラフィックスタイルはアクション再生で未使用のみ削除
  - 段落・文字スタイルは使用状況を取得できないため、強制ONのときだけ既定（先頭）以外を削除
- **パス／オブジェクト**：孤立点、空のテキスト、塗りも線もない（不可視）パス、不透明度0%、非表示オブジェクト、リンク切れの配置画像
- **グループ／レイヤー**：空のグループ／空のレイヤー・サブレイヤーを再帰削除
- **ガイド**（ラジオで1つ選択）：ロックされていないガイド／すべてのガイド（強制）／現在のアートボード以外
- **アートボード**：アートボード外・アクティブなアートボード外のオブジェクト、空のアートボード
- 「使用中の項目も削除（強制）」で、保護対象・既定以外まで踏み込んで削除
- 種類ごとに削除件数を集計して表示（0件の種類は省略）
- 日本語／英語UI

## 使い方

1. 整理したいドキュメントを開きます。
2. `AiDocumentCleaner.jsx` を実行します。
3. ダイアログで削除したい対象にチェックを入れます。
4. 未使用だけでなく保護対象・既定以外も削除したい場合は「使用中の項目も削除（強制）」をONにします。
5. ガイドはラジオで1つ選択します（既定は「削除しない」）。
6. 「削除」をクリックすると処理が実行され、種類ごとの削除件数が表示されます。

## 削除対象

| 分類 | 対象 |
| --- | --- |
| パネル項目 | スウォッチ／シンボル／ブラシ／グラフィックスタイル／段落スタイル／文字スタイル |
| パス／オブジェクト | 孤立点／空のテキスト／塗りも線もないパス／不透明度0%／非表示オブジェクト／リンク切れの配置画像 |
| グループ／レイヤー | 空のグループ／空のレイヤー・サブレイヤー |
| ガイド | ロックされていないガイド／すべてのガイド（強制）／現在のアートボード以外 |
| アートボード | アートボード外／アクティブなアートボード外のオブジェクト／空のアートボード |

## 注意事項

- 誤削除しやすい項目（非表示オブジェクト、リンク切れの配置画像、アートボード外／アクティブなアートボード外）は初期状態でOFFです。
- ガイドは既定で「削除しない」です。
- 段落スタイル・文字スタイルは使用状況を取得できないため、強制ON時のみ既定以外を削除します。
- グループ／レイヤーの掃除は他の削除より後に実行されるため、パスやオブジェクトの削除で空になった親も同じ実行で削除されます。
- 一時アクションは処理後に必ず解放・削除されます。
- クリッピングパスやコンパウンドパスの構成要素、ガイドは「塗りも線もないパス」の対象外です。
- ガイドは「ガイド」セクションでのみ削除されます（「塗りも線もないパス」やアートボード外の削除では対象になりません）。
- 実行前にファイルのバックアップを取ることをおすすめします。
