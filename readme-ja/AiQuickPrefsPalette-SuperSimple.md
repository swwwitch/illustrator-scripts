# クイック環境設定（SuperSimple）

[![Direct](https://img.shields.io/badge/Direct%20Link-AiQuickPrefsPalette--SuperSimple.jsx-ffcc00.svg)](https://github.com/swwwitch/illustrator-scripts/blob/master/jsx/preference/AiQuickPrefsPalette-SuperSimple.jsx)

[![Direct](https://img.shields.io/badge/Back%20to%20home-All%20scripts-cccccc.svg)](https://github.com/swwwitch/illustrator-scripts/blob/master/README.md)

---

「線幅と効果も拡大・縮小」を切り替えたい。「プレビュー境界」を一時的にオンにしたい。矢印キーの移動量を 1mm にしたい。

どれも作業中に何度も発生するのに、そのたびに環境設定ダイアログを開いて、目的のカテゴリを探して、チェックを入れて、［OK］。しかも項目は「一般」「単位」「選択範囲・アンカー表示」とバラバラの場所に散っていて、**普段よく触る数個だけが遠い**という状態です。

そこで、よく使う項目だけを1枚のパレットに集めて、クリックした瞬間に反映されるようにしました。開きっぱなしにしておく前提のパレットです。

この **SuperSimple** 版は、フル版（[AiQuickPrefsPalette](AiQuickPrefsPalette.md)）から変形（反転・回転）やアートボード関連のパネルを外し、環境設定の切り替えに絞った軽量版です。代わりに UI の明るさ（インターフェイスカラー）を切り替えるスウォッチと「ファイル管理を開く」ボタンを備えた「環境設定」パネルを持っています。

## 使い方

スクリプトを実行するとパレットが開きます。あとはチェックボックスやボタンを操作するだけで、**その時点で即座に反映**されます。

**［OK］も［適用］もありません。** 押した瞬間が確定です。パレットは `Esc` キー（パレットがアクティブなとき）でも閉じられます。

すでにパレットが開いている状態でもう一度実行すると、新しく開き直さずに既存のパレットを前面に出します。

## 操作できる項目

| パネル | 項目 | 内容 |
| --- | --- | --- |
| キー増加 | 数値＋単位 | 矢印キーでの移動量（環境設定 > 一般 > キー増加）。単位ポップアップで定規単位を切り替え |
| 整列オプション | プレビュー境界 | 整列・分布で線幅・効果を含む境界を使う |
| | 字形の境界に整列 | ポイント文字・エリア内文字を字形の境界で整列（2つまとめてON/OFF） |
| 変形オプション | パターン / 角 / 線幅と効果 | 変形時にパターンも変形、拡大・縮小時に角（ライブコーナー）の半径・線幅と効果も追従 |
| コピー/ペースト | 書式なしペースト / コピー元のレイヤーにペースト | |
| 描画 | リアルタイムの描画と編集 / プレビュー更新 | プレビュー更新は GPU プレビューの再描画ボタン |
| 環境設定 | インターフェイスカラー（4段階） | スウォッチをクリックすると環境設定（ユーザーインターフェイス）が開く |
| | ファイル管理を開く | 環境設定（ファイル管理）を開くボタン |

各項目にはツールチップが付いているので、どの環境設定に対応するかはポインタを重ねれば確認できます。

## Option+クリックで一括切替

「字形の境界に整列」と「変形オプション」のチェックボックスは、**Option+クリックでそのパネル内の全項目が同じ状態になります**。

- 「字形の境界に整列」：ポイント文字／エリア内文字の2つ
- 「変形オプション」：パターン／角／線幅と効果の3つ

通常のクリックはこれまでどおり個別切替です。「変形オプション」は3つまとめてオン／オフしたい場面が多いので、Option を添えるだけで済みます。

## キー増加の数値入力

数値欄では `↑` `↓` キーで増減できます。

| 操作 | 増減 |
| --- | --- |
| `↑` `↓` | ±1 |
| `Shift` + `↑` `↓` | ±10（次の10の倍数へスナップ） |
| `Option` + `↑` `↓` | ±0.1 |

負の値にはならず、0 でクランプされます。

単位ポップアップは表示単位を変えるだけのものではなく、**定規単位（rulerType）そのものを切り替えます**。ここが少し分かりにくいところです。単位を切り替えても、保存されている移動量の実寸（pt）は変わりません。表示だけが新しい単位で計算し直されます。

ポップアップに並ぶのは in / mm / pt / pica / cm / Q/H / px の7つです。

## 環境設定パネル（明るさ／ファイル管理）

### インターフェイスカラー（明るさ）

「環境設定」パネルには、暗／やや暗／やや明／明の4段階のスウォッチが並びます。現在の設定値にもっとも近い段階が選択枠（青）で表示されます。

インターフェイスカラーはスクリプトから直接は反映できないため、**スウォッチをクリックすると環境設定（ユーザーインターフェイス）が開きます**。開いたダイアログで矢印キー＋Return を押して確定してください。

### ファイル管理を開く

パネル下部の「ファイル管理を開く」ボタンで、環境設定（ファイル管理）を直接開きます。メニューコマンド `FilePref` を実行しています（Illustrator 2022 以降。旧「ファイル管理とクリップボード」統合パネルは `FileClipboardPref`）。

## 外部での変更に追従する

環境設定ダイアログなど、パレットの外で設定を変更した場合は、**パレットをクリック（再アクティブ）すれば表示が同期します**。

ただし、キー増加の数値欄にフォーカスがある間は同期しません。入力中の値が上書きされてしまうのを防ぐためです。

## 補足

Illustrator の常駐パレットは、表示している間に DOM への接続を失うという事情があるため、環境設定の**書き込み**はすべて BridgeTalk 経由でメインエンジンに委譲しています。一方、**読み出し**はエンジンを跨いでも安全なため、パレット側で直接・同期的に取得しています。

「角」だけは Boolean ではなく整数のプリファレンス（`policyForPreservingCorners`、1=ON / 2=OFF）なので、一括切替の共通処理には「値を書き込む関数」を項目ごとに渡す形にして吸収しています。

「プレビュー更新」ボタンは、メニューコマンド `View using GPU` を2回トグルして再描画を起こしています。
