# 選択オブジェクトを各アートボードの9点へ整列

[![Direct](https://img.shields.io/badge/Direct%20Link-AlignToArtboards.jsx-ffcc00.svg)](https://github.com/swwwitch/illustrator-scripts/blob/master/jsx/artboard/AlignToArtboards.jsx)

[![Direct](https://img.shields.io/badge/Back%20to%20home-All%20scripts-cccccc.svg)](https://github.com/swwwitch/illustrator-scripts/blob/master/README.md)

---

### 概要

- 複数アートボードのドキュメントで、選択オブジェクトを各アートボード上の指定位置へ整列するスクリプト。
- 3×3の9点から整列先を選び、左右・上下のマージンを指定できる。
- ダイアログ表示中はライブプレビューで結果を確認でき、キャンセルで元の位置に戻る。

### 主な機能

- 「整列の基準」を2種類から選択
  - **すべてのアートボード**：選択オブジェクトを中心点が属するアートボードごとに振り分け、各アートボード内の指定位置へ整列
  - **アクティブを基準**：アクティブアートボード上の選択を基準に、他のアートボード上の選択を同じ相対位置へ整列（アクティブ側は動かさない）
- 「整列先」は9点（左上／上中央／右上／左中央／中央／右中央／左下／下中央／右下）から選択
- 「マージン」で対応する辺から内側へオフセット。左右・上下を個別指定でき、「連動」ONで左右の値を上下にも反映（初期値：ON）
- マージンの入力単位はIllustratorのルーラー単位（rulerType）に連動し、内部でptに換算
- 「プレビュー境界を使用」で、線幅・効果を含む見た目の境界（ON）と図形本体の幾何境界（OFF）を切り替え
- キーボードショートカット
  - 整列先：q=左上 / w=上中央 / e=右上 / a=左中央 / s=中央 / d=右中央 / z=左下 / x=下中央 / c=右下
  - 整列の基準：1=すべてのアートボード / 2=アクティブを基準
  - マージン欄は↑↓で増減（Shift=±10、Option=±0.1）、Enter/ReturnでOK
- 日本語／英語の自動切り替え、各項目にツールチップ

### 処理の流れ

1. ドキュメントと選択の有無を確認
2. ダイアログで整列の基準・整列先・マージン・境界の種類を指定（変更のたびにプレビューを再適用）
3. 選択オブジェクトを中心点が属するアートボードごとに振り分け、指定アンカーへの移動量を計算
4. OKで確定、キャンセルまたは閉じる操作でプレビューの移動を巻き戻す

### 対象外

- ドキュメントが開かれていない場合、選択が無い場合
- ロック／非表示のオブジェクト、およびその状態の親レイヤー・親グループに含まれるオブジェクト
- 中心点がどのアートボードにも含まれないオブジェクト、bounds を取得できないオブジェクト
- 「中央」選択時のマージン、「アクティブを基準」選択時のマージンと整列先パネル（いずれも無効）
