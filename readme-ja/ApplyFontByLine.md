# 行のテキストをフォント名とみなして行単位でフォントを適用

[![Direct](https://img.shields.io/badge/Direct%20Link-ApplyFontByLine.jsx-ffcc00.svg)](https://github.com/swwwitch/illustrator-scripts/blob/master/jsx/fonts/ApplyFontByLine.jsx)

[![Direct](https://img.shields.io/badge/Back%20to%20home-All%20scripts-cccccc.svg)](https://github.com/swwwitch/illustrator-scripts/blob/master/README.md)

---

### 概要

- 選択したテキストフレームの各行（段落）について、その行の文字列をフォント名とみなして検索し、行単位で適用するスクリプト。
- フォント見本の作成に便利。厳密一致は自動適用し、あいまい一致・未一致だけを対話ピッカーで確認する。
- 適用できなかった行を含むフレームには、赤い半透明の目印を背面に作成する。

### 主な機能

- 選択物からテキストフレームを再帰収集（グループ内も対象。ロック・非表示は除外）
- 段階的なフォント照合
  - CUSTOM_MAP による特定文字列の強制割り当て（例：Jenson → Adobe Jenson Pro）
  - PostScript 名の完全一致、ファミリー名＋スタイル名の完全一致、ファミリー名の完全一致（厳密一致 → 自動適用）
  - ファミリー名への部分一致、フォント名全体への部分一致、先頭ワード（3文字以上）による一致（あいまい一致 → ピッカーの初期値）
  - 複数スタイルが候補になった場合は STYLE_PRIORITY（bold → semibold → medium → regular）の順で決定
  - 照合時は小文字化し、半角／全角スペースとピリオドを除去して表記ゆれを吸収
- フェーズ1では厳密一致のみをプログレスバー付きで自動適用
- フェーズ2で保留分をフォントピッカーで確認
  - **対象テキスト**表示、対象フレームへの自動ズーム（表示領域の60%に収まる倍率）
  - **検索**欄でフォントファミリーのプルダウンを絞り込み、**フォント**／**スタイル**を選択
  - **適用** / **スキップ** / **終了**（以降の保留をすべて未適用にして打ち切り）
  - 同じ文字列は実行中に再質問せず、選択結果を再利用
- 未適用の行を含むフレームの背面に、レイヤー「// missing-fonts」上へ赤・不透明度35%の長方形を作成し、レイヤーをロック（実行のたびに古い目印レイヤーは削除）
- 未適用文字列の一覧をダイアログで表示し、クリップボードへコピー可能
- 日本語／英語の自動切り替え

### 処理の流れ

1. 選択からテキストフレームを再帰収集し、フォント索引を作成
2. 各行を照合し、厳密一致は自動適用、それ以外は保留キューへ
3. 保留分をフォントピッカーで1件ずつ決定（スキップ・終了は未適用として記録）
4. 未適用フレームに目印を作成し、未適用文字列の一覧を表示

### 対象外

- ドキュメントが開かれていない場合（アラート）
- 選択が空、またはテキストフレームが1つも含まれない場合（アラート）
- ロックまたは非表示のオブジェクト（グループの場合は中身ごとスキップ）
- 空行・空白のみの行
- フォントの適用のみを行い、テキストの内容やサイズは変更しない
- モーダルダイアログ表示中のライブプレビューは（クラッシュ対策のため）行わない
