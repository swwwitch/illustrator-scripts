# 選択したテキストをアーチ状のパス上文字に変換

[![Direct](https://img.shields.io/badge/Direct%20Link-ArcTextGenerator.jsx-ffcc00.svg)](https://github.com/swwwitch/illustrator-scripts/blob/master/jsx/text/ArcTextGenerator.jsx)

[![Direct](https://img.shields.io/badge/Back%20to%20home-All%20scripts-cccccc.svg)](https://github.com/swwwitch/illustrator-scripts/blob/master/README.md)

---

### 概要

- 選択したポイント文字／パス上文字から、その文字幅に合わせたアーチ状のパスを自動生成し、パス上文字に変換するスクリプト。
- カーブの深さ・向き・パス上文字の効果をダイアログで指定でき、プレビューで確認しながら調整できる。
- 生成されるアーチパスは塗り・線なし（不可視）なので、文字だけが見える状態になる。

### 主な機能

- **カーブ：** スライダー（0〜100、初期値50）でアーチの深さを調整。0で直線に近く、100で最も丸くなる
- **向き：** アーチの膨らむ方向を「上向き」（初期値）／「下向き」から選択
- **パス幅に合わせる：** 「しない」（初期値）／「文字サイズ」／「トラッキング」から選択
  - 文字サイズ：文字サイズを変えてパス両端まで収める（0.1pt刻みで縮小、必要なら一度拡大してから調整）
  - トラッキング：文字サイズを保ったまま、字間を粗調整→微調整してパス両端に合わせる
- **効果：** パス上文字オプションの効果を「虹」（初期値）／「歪み」／「3D リボン」／「階段」／「引力」から選択
- **トラッキング：** チェックONのときのみ、既存のトラッキング値に指定値を加算（−100〜500、矢印キーで増減：Shiftで10、Optionで0.1）。OFFにすると0に戻る。「パス幅に合わせる：トラッキング」を選択中はディム表示
- **プレビュー：** 初期値ON。ONの間は一時オブジェクトで結果を表示し、OFFまたはキャンセルで元に戻る
- 行揃えは常に中央揃え
- 日本語／英語の自動切り替え、各項目にツールチップ

### 処理の流れ

1. 選択からポイント文字／パス上文字を収集（グループ内も再帰的に探索）
2. テキストを一時的にアウトライン化して実寸を計測し、ベースライン上に直線パスを作成
3. ベジェハンドルを操作して直線をアーチ状に変形し、パス上文字を生成して元テキストの文字属性を複製
4. 中央揃え・トラッキング・効果を適用し、選択されたフィット方法でパス幅に合わせる（元のテキストと同時選択されていたパスは削除）

### 対象外

- ドキュメントが開かれていない場合
- 選択にポイント文字・パス上文字が含まれていない場合（エリア内文字は対象外）
- 空のテキスト、行や文字範囲を取得できないテキスト
- フィット処理は閉じたパス上の文字、ロック／非表示／編集不可のテキストには適用されない
