# アートボード単位でオブジェクトをレイヤーに整理

[![Direct](https://img.shields.io/badge/Direct%20Link-ArtboardLayerOrganizer.jsx-ffcc00.svg)](https://github.com/swwwitch/illustrator-scripts/blob/master/jsx/layers/ArtboardLayerOrganizer.jsx)

[![Direct](https://img.shields.io/badge/Back%20to%20home-All%20scripts-cccccc.svg)](https://github.com/swwwitch/illustrator-scripts/blob/master/README.md)

---

### 概要

- ドキュメント内のオブジェクトをアートボード単位で振り分け、「番号_アートボード名」のレイヤーに整理するスクリプト。
- 所属アートボードは各オブジェクトの重心位置で判定する。
- ガイドは「_guide」、どのアートボードにも属さないオブジェクトは「_pasteboard」へ集約する（必要時のみ生成）。
- 整理後のレイヤー順はアートボード順（上から 1→2→3…）に揃える。

### 主な機能

- 「対象のアートボード」を選択
  - **現在のアートボードのみ** / **すべて**（アートボードが1つのときは「現在のアートボードのみ」に固定）
- 「レイヤー名」の構成を指定
  - 「アートボード番号」「アートボード名」の各要素をON/OFF
  - 「区切り文字」をアンダースコア（_）／ハイフン（-）／半角スペース／なしから選択（既定：アンダースコア）
  - アートボード名が空のときは「アートボード」を代替名として使用
- 「対象外にする」で除外条件を指定
  - ロック：レイヤー／オブジェクト、非表示：レイヤー／オブジェクト（いずれも既定ON）
  - 「指定レイヤー」欄にレイヤー名をカンマまたは「、」区切りで入力して整理対象外に（既定：bg）
  - ガイドは「指定レイヤー」除外を無視して _guide へ集約（ロック／非表示の祖先は尊重）
- 「後処理」で空のレイヤー／サブレイヤーの削除をON/OFF（既定ON）
- 旧仕様レイヤー（アートボード名のみのレイヤー）を新仕様レイヤーへ統合して削除
- 移動先レイヤーがロック／非表示でも自動で解除して移動し、処理後に元の状態へ戻す
- 「_guide」「_pasteboard」は空でも削除しない保護レイヤー。_guide は最前面へ移動
- 移動に失敗したオブジェクトがある場合は件数をアラート表示
- 日本語／英語の自動切り替え

### 処理の流れ

1. ダイアログで対象アートボード・レイヤー名・除外条件・後処理を指定
2. トップレベルのオブジェクトを収集し、重心位置から所属アートボードを判定してレイヤーへ移動
3. 対象外のオブジェクトを _pasteboard／_guide へ振り分け、旧仕様レイヤーを統合
4. レイヤー順をアートボード順に整え、必要に応じて空のレイヤー／サブレイヤーを削除

### 対象外

- ドキュメントが開かれていない場合（警告を表示して終了）
- 除外設定ONのときのロック／非表示のレイヤー・オブジェクト、およびそれらを祖先に持つオブジェクト
- 「指定レイヤー」で指定した名前のレイヤーとその配下（ガイドを除く）
- 「現在のアートボードのみ」選択時の _pasteboard 振り分けと旧仕様レイヤーの統合（実行しない）
- 保護レイヤー（_guide／_pasteboard）は空でも削除しない
