# あふれたテキストの文字サイズ・エリア高さを自動調整

[![Direct](https://img.shields.io/badge/Direct%20Link-AutoFitTextFrame.jsx-ffcc00.svg)](https://github.com/swwwitch/illustrator-scripts/blob/master/jsx/text/AutoFitTextFrame.jsx)

[![Direct](https://img.shields.io/badge/Back%20to%20home-All%20scripts-cccccc.svg)](https://github.com/swwwitch/illustrator-scripts/blob/master/README.md)

---

### 概要

- 選択したエリア内文字・パス上文字のオーバーセット（文字あふれ）を、文字サイズまたはフレームの高さを調整して解消するスクリプト。
- 「あふれをなくす」だけでなく、余白がある場合に「あふれない最大サイズまで詰める」処理も選べる。
- 変数／データセットを使ったドキュメントでは、元の値をタグに記録して各データセット適用時にリセットしてから処理する。

### 主な機能

- **文字サイズ：あふれ処理**（初期値ON）：オーバーセットが解消されるまで文字サイズを0.1pt刻みで縮小
- **文字サイズ：ぴったり**（初期値ON）：いったん文字サイズを倍々に拡大してオーバーセットを発生させ、その後あふれ処理で詰める（余白があっても最大サイズまで詰まる）
- 両方ONのときは「ぴったり」→「あふれ処理」の順で実行
- **エリア内文字の高さ調整**：選択にエリア内文字があるときのみ有効。ONにすると文字サイズの処理は無効になり、以下のオプションから選ぶ
  - **高さを調整**：自動サイズ調整を一時的にON→OFFにして、必要な分だけ高さを拡張して固定
  - **自動サイズ調整**：エリア内文字に自動サイズ調整を適用（拡張のみ・OFFには戻さない）
- 手動行送りが設定されている場合、文字サイズの変更に合わせて行送りを同じ比率で追従
- エリア内文字は Illustrator の `overflows` 判定を優先し、パス上文字は表示行に収まる文字数で判定
- 選択したグループ内のテキスト、テキスト範囲（カーソル選択）も対象
- ダイアログ位置を記憶、日本語／英語の自動切り替え

### 処理の流れ

1. 選択がない場合は警告して終了。選択からエリア内文字・パス上文字を再帰的に収集（重複は除去）
2. データセットの1件目なら、元の文字サイズ／高さをタグに記録し、各回の処理前にリセット
3. 選択した処理（ぴったり／あふれ処理／高さ調整／自動サイズ調整）を対象テキストに適用
4. 最終データセットの処理後にタグを削除

### 対象外

- ドキュメントが開かれていない場合、および何も選択されていない場合
- ポイント文字（エリア内文字・パス上文字のみが対象）
- ロック／非表示／編集不可のテキスト
- 改行コードを含むテキスト（文字サイズ処理時に警告を表示して中止）
- 高さ調整・自動サイズ調整は、選択にエリア内文字がない場合は実行されない
