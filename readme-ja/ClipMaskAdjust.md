# クリップグループのマスクと内容を調整

[![Direct](https://img.shields.io/badge/Direct%20Link-ClipMaskAdjust.jsx-ffcc00.svg)](https://github.com/swwwitch/illustrator-scripts/blob/master/jsx/mask/ClipMaskAdjust.jsx)

[![Direct](https://img.shields.io/badge/Back%20to%20home-All%20scripts-cccccc.svg)](https://github.com/swwwitch/illustrator-scripts/blob/master/README.md)

---

### 概要

- クリップグループ（クリッピングマスク）の「マスクパス」と「内容」をまとめて調整するスクリプト。
- ダイアログの操作はオートプレビューで即時カンバスに反映される。
- 数値は定規の単位（rulerType）に合わせて表示・入力できる。
- 安全重視のためUndo／ヒストリー操作は未実装。キャンセルしてもプレビューで反映された状態は戻らない。

### 主な機能

- 「基準点」：9点のラジオボタンで内容の揃え位置を指定
- 「微調整」：X／Y に現在の単位で数値を入力し、基準点からの位置をずらす
- 「フィットとスケール」
  - 縦横比を保持して切り取り（マスクを覆うように拡大／縮小）
  - 縦横比を保持して縮小（マスクに収まるように拡大／縮小）
  - サイズ保持（内容の大きさは変えず位置合わせのみ）
  - スケールを指定（％で直接入力。現在のスケールが初期値として入る）
- 「マスクパス」：そのまま／内容に合わせる／正方形に（短辺基準で中央から作成）
- 「角丸」：クリップグループ全体に「角を丸くする」効果を適用。初期値は（マスクの幅＋高さ）÷25 を現在の単位で切り上げた値
- 「正円」：「正方形に」を選んでいるときのみ有効。ONにすると角丸を自動でONにし、半径を短辺の半分に設定
- 角丸の値を変更したときは既存の効果を消去してから再適用し、効果の重複を防止
- 基準点はキーボード（q/w/e、a/s/d、z/x/c）で切り替え可能
- 数値入力欄は ↑↓ で±1、Shift+↑↓ で±10、Option+↑↓ で±0.1
- 日本語／英語の自動切り替え

### 処理の流れ

1. 選択中のクリップグループからマスクパスと内容を分離
2. マスクパスの形状を指定（内容に合わせる／正方形に）に応じて変更
3. 「角丸」がONならクリップグループに角を丸くする効果を適用
4. マスクパスの visibleBounds を基準に内容を拡大／縮小し、基準点と微調整値で位置合わせ

### 対象外

- オブジェクトが未選択の場合（アラートを表示して終了）
- クリップグループ（clipped な GroupItem）以外のオブジェクト
- マスクパスが見つからない、または内容が空のクリップグループ
- Undo／ヒストリーによる復元（キャンセルしてもプレビューの結果は残る）
