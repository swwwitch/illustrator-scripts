# 選択オブジェクトの色からスウォッチとグラデーションを自動生成

[![Direct](https://img.shields.io/badge/Direct%20Link-CreateGradientFromSelection.jsx-ffcc00.svg)](https://github.com/swwwitch/illustrator-scripts/blob/master/jsx/color/CreateGradientFromSelection.jsx)

[![Direct](https://img.shields.io/badge/Back%20to%20home-All%20scripts-cccccc.svg)](https://github.com/swwwitch/illustrator-scripts/blob/master/README.md)

---

### 概要

- 選択オブジェクトの塗り／線カラーを配置順（横並びなら左→右、縦並びなら上→下）で抽出し、スウォッチグループに登録するスクリプト。
- 抽出した色から線形グラデーションを作成し、長方形に適用したりグラフィックスタイルとして登録できる。
- 選択が空でも、スウォッチパネルで2つ以上のスウォッチを選択していればそれを入力として使える。

### 主な機能

- グループ／複合パス／テキストを再帰的に走査し、塗りと線の両方から色を収集（重複色は除外）
- **グローバルカラー化**：スウォッチをグローバル（プロセス）カラーとして登録し、あとから一括で色を変更可能にする
- **グラデーションを作成**：抽出色を各ストップに割り当てた線形グラデーションを生成
  - **通常**：色を等間隔に配置したスムーズなグラデーション
  - **セパレート**：境界をくっきり分割したグラデーション（2〜6色。選択が7つ以上のときは選択不可）
- **長方形を作成してグラデーションを適用**：作成したグラデーションを塗りにした長方形を配置
  - **選択オブジェクトのサイズに合わせる**：選択範囲の外接サイズで長方形を作成（オフのときは100pt角、スウォッチ由来のときは200×100pt固定）
  - 横並びの選択なら左端を揃えて真下、縦並びなら上端を揃えて右側に配置
- **グラフィックスタイルとして登録**：長方形の見た目をグラフィックスタイルに登録（長方形の作成がオフのときは一時レイヤー上の一時長方形を使い、登録後に削除）
- 選択が縦並びと判定された場合は、アクション（gradient/90degree）でグラデーション角度を90°にする
- 日本語／英語の自動切り替え、主要項目にツールチップ
- ダイアログの設定値は targetengine 内でセッション保持（Illustrator の再起動で消える）

### 処理の流れ

1. 選択オブジェクト（または選択スウォッチ）から色を収集し、並び方向を判定
2. ダイアログでカラー・長方形まわりのオプションを指定
3. 新規スウォッチグループ（AutoGradient）を作成し、抽出色を AutoColor として登録
4. グラデーションを作成し、必要に応じて長方形の配置とグラフィックスタイル登録を実行

### 対象外

- ドキュメントが開かれていない場合（無言で終了）
- 抽出できた色が1色以下の場合（無言で終了）
- 選択が空で、かつ選択スウォッチが2つ未満の場合
- 塗りも線も「なし」のオブジェクト、bounds を取得できないオブジェクト
- ロックまたは非表示のレイヤーしかない場合、長方形は作成されない
