# 塗りと線の入れ替え・変換・消去

[![Direct](https://img.shields.io/badge/Direct%20Link-FillStrokeSwitcher.jsx-ffcc00.svg)](https://github.com/swwwitch/illustrator-scripts/blob/master/jsx/color/FillStrokeSwitcher.jsx)

### 概要

- 選択したオブジェクトの塗りと線をまとめて調整するスクリプト。
- 「変換」と「消去」の2パネルから7つの処理モードをラジオボタンで選択できる。
- プレビューを見ながらモードを切り替えられ、キャンセルで元の見た目に復元される。

### 主な機能

- 変換モード
  - **塗り↔線**：オブジェクトごとに塗りと線を入れ替える
  - **塗り→線**：塗りの色を線に適用（線幅がなければ 1 を補完）
  - **線→塗り**：線の色を塗りに適用
  - **2つのオブジェクト間で交換**：選択した2つのオブジェクトの塗り・線の見た目を交換（選択が2つのときのみ有効・かつ初期選択）
- 消去モード
  - **塗りを消去** / **線を消去** / **塗りと線を消去**
- **プレビュー**：結果を一時的に反映し、キャンセルで元の見た目に戻す
- グラデーション、複合パス、グループに対応（グループ・複合パスは中身を再帰処理）
- テキスト（ポイント文字、エリア文字、パス上文字）は文字単位で処理し、部分的な文字色を維持
- OK 後に元の選択を復元
- 処理に失敗したパス／テキスト／選択復元があれば、件数と詳細（最大8件）をアラートで表示
- 日本語／英語の自動切り替え、各モードにツールチップ

### 処理の流れ

1. ドキュメントと選択（1〜2オブジェクト）を検証
2. 選択の見た目をスナップショットし、モード選択ダイアログを表示
3. プレビュー時はモード切り替えのたびに復元→再適用
4. OKで確定（プレビュー未適用なら本適用）、キャンセルで元の見た目に復元

### 対象外

- ドキュメントが開かれていない場合（アラート）
- 選択が0個、または3個以上の場合（アラート）
- パス・複合パス・グループ・テキスト以外のオブジェクト（配置画像、シンボルなど）は処理されない
- 「2つのオブジェクト間で交換」は PathItem／CompoundPathItem／TextFrame のみが対象
