# アートボード名を解析して行列グリッドに再配置

[![Direct](https://img.shields.io/badge/Direct%20Link-GridArrangeArtboards.jsx-ffcc00.svg)](https://github.com/swwwitch/illustrator-scripts/blob/master/jsx/artboard/GridArrangeArtboards.jsx)

[![Direct](https://img.shields.io/badge/Back%20to%20home-All%20scripts-cccccc.svg)](https://github.com/swwwitch/illustrator-scripts/blob/master/README.md)

---

### 概要

- アートボード名を解析し、行列グリッドとしてアートボードを再配置するスクリプト。
- 名前全体が「行番号＋区切り文字（-, _, x）＋列番号」に一致するものを行列指定として扱う（例：1-1、1_2、2x1）。
- 行列指定に一致しない場合のみ「接頭辞＋区切り文字＋番号」を接頭辞指定として扱い（例：banner-1、iconx3）、通常の行列指定の下に接頭辞ごとの行として配置する。
- アートボード上のオブジェクトも、同じ移動量で一緒に移動する。

### 主な機能

- 「間隔」パネルで列間・行間を指定。「連動」ONで列間の値を行間にも反映
  - 入力単位は環境設定の定規単位（rulerType）に連動
  - 初期値はアクティブアートボード幅の1/8を整数に丸めた値
  - ↑↓キーで増減（Shift=10の倍数にスナップ、Option=±0.1）
- 「未指定／重複」の例外処理を2種類から選択
  - **直前の行の末尾に配置**：非一致名を直前にマッチした行の末尾（最大列＋1以降）へ並べる
  - **最後の行にまとめる**：非一致名を最終行に出現順で並べる
  - 例外領域に入る境界では間隔を3倍に広げて区別しやすくする
- 「ロック／非表示を除外」で、レイヤー単位・オブジェクト単位に移動対象から除外
- 「パネル上の並び順を変更」ONで、アートボードパネルの並びを行→列の昇順に再構築
- 列幅・行高は「列ごと・行ごとの最大サイズ」で計算するため、サイズの異なるアートボードが混在しても重ならない
- 処理後に「すべてのアートボードを全体表示」を実行
- 日本語／英語の自動切り替え

### 処理の流れ

1. アートボード名を走査し、行列指定・接頭辞指定・非一致を判定
2. ダイアログで間隔・例外処理・除外設定・並び順変更を指定
3. 行・列番号を割り当て、列ごとの最大幅・行ごとの最大高さから累積オフセットを計算
4. アートボード上のオブジェクトを先に移動し、続いてアートボード矩形を更新（必要ならパネル順も再構築）

### 対象外

- ドキュメントが開かれていない場合
- 「行-列」「接頭辞-番号」形式の名前が1つも見つからない場合（警告を表示して終了）
- 0行・0列・0番（無効として扱う）
- 数字だけの接頭辞（行列指定との混同を避けるため接頭辞指定にしない）
- グループ内の個別オブジェクト（グループ単位で移動）
- 中心が元のアートボード矩形の外にあるオブジェクト
- 除外設定ONのときのロック／非表示レイヤー・オブジェクト
