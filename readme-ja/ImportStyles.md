# 登録済みAIファイルからスタイル・ブラシ・フォント見本を読み込み

[![Direct](https://img.shields.io/badge/Direct%20Link-ImportStyles.jsx-ffcc00.svg)](https://github.com/swwwitch/illustrator-scripts/blob/master/jsx/style/ImportStyles.jsx)

[![Direct](https://img.shields.io/badge/Back%20to%20home-All%20scripts-cccccc.svg)](https://github.com/swwwitch/illustrator-scripts/blob/master/README.md)

---

### 概要

- あらかじめ登録しておいた AI ファイルを一覧から選び、その中身を現在のドキュメントへ取り込むスクリプト。
- グラフィックスタイル、ブラシ、シンボル、フォント見本などを、オブジェクトごと貼り付けて持ち込める。
- 候補リストは TSV ファイル（`ImportStyles_candidates.tsv`）に永続化され、ダイアログから追加できる。

### 主な機能

- 「コンテンツ」「ファイル名」の2列＋ヘッダ付き ListBox（スクロール可）で候補を表示
- **カテゴリー**ラジオ（**スタイル／ブラシ／シンボル** / **フォント**）で候補を絞り込み（初期状態は全件表示）
- 検索欄＋**検索**ボタンで、コンテンツ名とファイル名を対象に大文字小文字を無視して絞り込み（ボタン押下時のみ適用）
- ダブルクリックまたは Enter キー、**読み込み**ボタンで選択を確定
- **追加**ボタン：任意の AI ファイルを選んでスタイル用フォルダへコピーし、表示名とカテゴリー（ファイル名から自動推定）を指定して候補 TSV に登録／上書き
- 貼り付け先は「// _imported」レイヤー（無ければ自動作成、ロック解除・表示状態にする）
- 候補 TSV は `label / path / category` の3列。旧形式の check 列（0|1）は読み込み時に無視
- TSV が存在しない場合は既定候補（オープンパス、矢印、丸数字、定番フォント（欧文））を使用
- 日本語／英語の自動切り替え

### 処理の流れ

1. ダイアログで候補から AI ファイルを選択
2. 対象 AI を開き、作業アートボード内のオブジェクトをすべてコピー
3. 対象 AI を保存せずに閉じる
4. 元ドキュメントへ復帰し、取り込み先レイヤー「// _imported」を用意して貼り付け

### 対象外

- 元のドキュメントが開かれていない場合（アラートを表示して終了）
- 候補のパスにファイルが存在しない場合（アラートを表示して終了）
- 作業アートボードの外にあるオブジェクトはコピーされない
- Illustrator の仕様により、グラフィックスタイル／ブラシは最低1件が残る
