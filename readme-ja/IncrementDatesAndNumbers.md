# テキスト内の日付・曜日・数値を一括で増減

[![Direct](https://img.shields.io/badge/Direct%20Link-IncrementDatesAndNumbers.jsx-ffcc00.svg)](https://github.com/swwwitch/illustrator-scripts/blob/master/jsx/text/IncrementDatesAndNumbers.jsx)

[![Direct](https://img.shields.io/badge/Back%20to%20home-All%20scripts-cccccc.svg)](https://github.com/swwwitch/illustrator-scripts/blob/master/README.md)

---

### 概要

- 選択したテキスト内に含まれる日付・曜日・時刻・連番・数値などをまとめて増減するスクリプト。
- 日付を増減すると、かっこ内の曜日も自動で追従する。
- ダイアログで「オリジナル」と「結果」を並べて確認しながら増減値を決められる。

### 主な機能

- **増減：値**：加算・減算する量を指定（整数のみ、初期値1）。矢印キーで増減（Shiftで10、Optionで0.1）、0を指定すると変化しない
- **対象**：年月日を含むテキストのときだけ表示され、選択中の実際の値（例「2025」「11」「21」）がラジオボタンのラベルになる。年／月／日のどれを増減するかを選択（初期値は日）。切り替えると増減値は1にリセットされる
- **種別**：「12.1」のようなドット区切り2要素のときだけ表示され、「数字」（初期値）／「日付」の解釈を選択
- ダイアログ上部にオリジナルと結果をリアルタイム表示（数値の場合は計算結果も併記）
- 対応する主な文字列パターン
  - 和文日付：2025年11月21日、2025年11月21日（金）、11月21日㊎
  - 元号付き日付：令和7年3月21日（金）など（改元をまたぐ場合は元号も更新）
  - スラッシュ／ドット区切り：2025/11/21、2025/11/21（Fri）、2025.11.21、29.3.2、29.3
  - 8桁の日付：20251121
  - 時刻：19:00（「年」対象で時、それ以外で分を増減。分の繰り上げ／繰り下げと24時間ループに対応）
  - 曜日単体：日〜土、Sun〜Sat、㊐〜㊏
  - 西暦（年のみ）：2025 など
  - 一般的な数値：123、1,234、100.5（小数は桁数を維持して最小桁を増減、カンマ区切りも維持）
- 選択した複数のテキスト、グループ内のテキストも一括処理（複数行は行ごとに処理）
- 日本語／英語の自動切り替え

### 処理の流れ

1. 選択の先頭にあるテキストを解析し、年月日・ドット2要素などのパターンを判定してダイアログの表示内容を決定
2. 増減値・対象・種別を指定（変更のたびに結果プレビューを更新）
3. OKで選択中のテキストを走査し、1行ずつパターンに応じて日付・曜日・時刻・数値を増減
4. 更新した文字列をテキストフレームに書き戻す

### 対象外

- 小数の増減値（整数のみ許容）
- テキストフレーム以外のオブジェクト（グループは中のテキストを再帰的に処理）
- 1行につき最初にマッチしたパターンのみを処理（同一行に複数の日付・数値がある場合、2つ目以降は対象外）
