# テキストの文字列でグラフィックスタイルを登録

[![Direct](https://img.shields.io/badge/Direct%20Link-RegisterGraphicStyleWithText.jsx-ffcc00.svg)](https://github.com/swwwitch/illustrator-scripts/blob/master/jsx/style/RegisterGraphicStyleWithText.jsx)

[![Direct](https://img.shields.io/badge/Back%20to%20home-All%20scripts-cccccc.svg)](https://github.com/swwwitch/illustrator-scripts/blob/master/README.md)

---

グラフィックスタイルを登録するとき、地味に面倒なのが**名前をつける工程**です。

パネルの新規ボタンで登録すると「グラフィックスタイル 1」のような名前がつくので、ダブルクリックして名前を入力し直す。スタイルが10個あれば10回。しかもスタイル名を考えながら作業していると、たいていアートボード上には「見本の図形」と「その名前を書いたテキスト」が並んでいたりします。

**その並んでいるテキストを、そのままスタイル名にすればいいのでは。**

そう考えて作ったのがこのスクリプトです。図形とテキストを一緒に選んで実行すると、図形の見た目がグラフィックスタイルとして登録され、テキストの文字列がそのままスタイル名になります。

<img alt="" src="" width="50%" />

## 使い方

1. 見た目を登録したいオブジェクトと、スタイル名にしたいテキストを選択
2. スクリプトを実行

ダイアログはありません。実行するとそのまま登録されます。

## 選択のしかた

| 選択 | 動作 |
| --- | --- |
| テキスト1点＋オブジェクト1点 | その組み合わせで1つ登録 |
| グループ1点（中身がテキスト＋オブジェクト） | グループの中身を見て1つ登録 |
| 複数のグループ | グループごとに登録（まとめて処理） |

いちばん便利なのは3つ目です。「図形＋名前のテキスト」をグループにしたものをアートボードに並べておき、全部選んで一度実行すれば、スタイルが一気に登録されます。スタイルの一覧表をそのままパネルに流し込むような使い方ができます。

上記のいずれにも当てはまらない選択のときは、何もせずに終了します。テキストの中身が空（または空白だけ）の場合、そのジョブはスキップされます。

## 同名スタイルは上書き

登録しようとしているスタイル名と同じ名前のスタイルが既にある場合は、**先に削除してから登録し直します**。

図形の色を調整して再実行、また調整して再実行、という試行錯誤をしても「スタイル名 1」「スタイル名 2」のように増えていきません。そのまま何度でも上書きできます。

## テキストの見た目は登録されない

登録の直前に、**スタイル対象のオブジェクトだけを選択し直しています**。

テキストと図形の両方を選んだまま登録すると、テキストの見た目まで混ざったスタイルになってしまうためです。あくまでテキストは「名前を提供する係」で、スタイルになるのは図形のほうだけです。

## 対象

TextFrame（スタイル名の供給元）と、それ以外のページアイテム（スタイルの登録対象）。グループ選択の場合は、グループ直下の2点を仕分けます。

## 補足

グラフィックスタイルの新規登録は、通常 `graphicStyles.add()` のような API で行いたいところですが、Illustrator のグラフィックスタイルには追加用のメソッドがありません。

そこで、「新規グラフィックスタイル」を記録した一時アクション（`.aia`）を `Folder.temp` に書き出して `app.loadAction()` でロードし、`app.doScript()` で再生する方式をとっています。アクションはジョブごとではなく**最初に一度だけロード**し、全ジョブの処理が終わってから `app.unloadAction()` でアンロードします。

アクション再生後は、`graphicStyles.length` が増えたことを確認してから末尾のスタイルを改名しています。登録が行われなかった場合に、無関係な既存スタイルの名前を書き換えてしまわないためです。

処理が終わると、実行前の選択状態に戻します。
