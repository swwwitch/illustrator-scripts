# 選択オブジェクトの回転を水平（0°）に補正

[![Direct](https://img.shields.io/badge/Direct%20Link-ResetRotation.jsx-ffcc00.svg)](https://github.com/swwwitch/illustrator-scripts/blob/master/jsx/transform/ResetRotation.jsx)

[![Direct](https://img.shields.io/badge/Back%20to%20home-All%20scripts-cccccc.svg)](https://github.com/swwwitch/illustrator-scripts/blob/master/README.md)

---

### 概要

- 選択オブジェクトの見かけ上の回転を水平（0°）に補正するスクリプト。
- テキスト、配置画像（リンク／埋め込み）、長方形（パス）が対象。
- 画像がクリッピンググループ内にある場合は、回転が乗っている親（ホスト）を自動検出して補正。
- 縦書きテキストはスキップし、しきい値内のほぼ水平なオブジェクトは処理しない。

### 主な機能

- 「対象」パネル：テキスト／配置画像（埋め込み・リンク）／長方形（パス）のチェックボックス（いずれもデフォルトON）
- 「クリップグループ」チェックボックス：ONのときはクリップグループ自体を回転（範囲は「最上位のクリップ」に固定。UIからは選択できない）
- 「テキスト」パネルの「縦横比を正す」：テキストの文字水平比率・垂直比率を100%に戻す（デフォルトON）
- 「補正条件」パネルの「水平とみなす範囲(°)」：しきい値を数値入力（0.01〜10° にクランプ、初期値0.1）。↑↓で±1、Shift+↑↓で±10、Option+↑↓で±0.1
- 変換行列から回転角を推定し、パスは頂点（最初のセグメント）からの推定にフォールバック
- 鏡像（行列式が負の反転）を考慮して回転方向を決定
- 回転はオブジェクト中心（Transformation.CENTER）で実行し、直後に各オブジェクトへ個別に「バウンディングボックスをリセット」を実行
- 同一ホストが複数回対象になる場合は一度だけ処理
- 日本語／英語の自動切り替え

### 処理の流れ

1. 選択オブジェクトを再帰走査し、テキスト／画像／長方形をタイプ別に収集
2. 変換行列（またはパス頂点）から回転角を推定し、しきい値内なら「すでに水平」としてスキップ
3. 鏡像を考慮して必要な角度だけ、オブジェクト中心を基準に回転
4. 回転直後に対象を選択して「バウンディングボックスをリセット」を実行し、元の選択に復帰

### 対象外

- ドキュメントが開いていない場合、オブジェクトが未選択の場合
- 縦書きテキスト（クリップグループ外のテキストフレームのみ判定）
- 4頂点の閉じたパス以外のパス（長方形として扱われない）
- しきい値（水平とみなす範囲）以内のほぼ水平なオブジェクト
- クリップグループモードのときのマスクパス自体
- ロック／非表示などで回転できないオブジェクト（例外は無視してスキップ）
