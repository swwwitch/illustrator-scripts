# 選択オブジェクトを条件で絞り込み再選択

[![Direct](https://img.shields.io/badge/Direct%20Link-SmartSelectionFilter.jsx-ffcc00.svg)](https://github.com/swwwitch/illustrator-scripts/blob/master/jsx/select/SmartSelectionFilter.jsx)

[![Direct](https://img.shields.io/badge/Back%20to%20home-All%20scripts-cccccc.svg)](https://github.com/swwwitch/illustrator-scripts/blob/master/README.md)

---

### 概要

- 選択中のオブジェクトから、条件に合うテキストやパスだけを残して選択し直すスクリプト。
- テキストの種類・行揃え・使用フォント、パスの開閉や塗り／線の状態を条件にできる。
- チェックボックスの変更はダイアログを閉じずにカンバスへ反映される。
- キャンセルすると、開始前の選択状態と表示状態に戻る。

### 主な機能

- 「テキスト」パネル：ポイント文字／エリア内文字／パス上文字（それぞれ件数を表示）
- 「行揃え」パネル：左揃え／中央揃え／右揃え（テキストが1つも選ばれていないときは無効化）
- 「フォント」パネル：選択内で使われているフォントを一覧化し、フォント単位で絞り込み（件数付き、フォントがなければ空）
- 「パス」パネル：オープンパス／水平線／垂直線（幅または高さがほぼ0のオープンパスを水平線・垂直線として判定）
- 「クローズパス」パネル：塗りのみ／線のみ／塗りと線
- 「まとめて選択」パネル：テキスト／ケイのみ／塗りのみ を1クリックで指定。詳細条件を変更すると、こちらのチェック状態も自動で同期
- 「対象外オブジェクト」パネル：何もしない／非表示／不透明度（スライダーで0〜100%、ラジオのラベルに現在値を表示）。スライダーは不透明度を選んでいるときのみ有効
- Option（Alt）を押しながらチェックボックスをクリックすると、すべてのチェックボックスを同じ状態に一括切り替え
- アウトライン表示／プレビュー表示を切り替えるボタン（終了時には元の表示に戻る）
- 日本語／英語の自動切り替え

### 処理の流れ

1. ドキュメントと選択の有無を確認し、選択を配列として退避
2. 選択内（グループの中身も再帰的に）を走査して、種類・行揃え・フォント・パス状態ごとの件数を集計
3. ダイアログで条件を切り替えるたびに選択と対象外オブジェクトの表示状態を再計算し、カンバスへ即時反映
4. OKで現在の選択状態を確定、キャンセルで開始前の選択状態と表示状態に復元

### 対象外

- ドキュメントが開かれていない場合、オブジェクトが未選択の場合
- テキストフレームとパス以外のオブジェクト（グループは中身を再帰的に対象とするが、グループ自体は残らない）
- 塗りも線も設定されていないクローズパス
