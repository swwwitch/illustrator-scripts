# 2つのオブジェクトの位置を入れ替え

[![Direct](https://img.shields.io/badge/Direct%20Link-SwapObjects.jsx-ffcc00.svg)](https://github.com/swwwitch/illustrator-scripts/blob/master/jsx/alignment/SwapObjects.jsx)

[![Direct](https://img.shields.io/badge/Back%20to%20home-All%20scripts-cccccc.svg)](https://github.com/swwwitch/illustrator-scripts/blob/master/README.md)

---

### 概要

- 選択した2つのオブジェクトの位置を入れ替えるスクリプト。
- 幅が異なるオブジェクトでも、2つ並び全体の左端・右端を保ったまま入れ替えられる。
- プレビューを見ながら基準を切り替えられ、キャンセルで元の位置に戻る。

### 主な機能

- 位置の基準を2種類から選択
  - **中心位置を交換**：お互いの中心が入れ替わる（幅が異なる場合、全体の左右端は移動する）
  - **両端の位置を保って交換**：左端・右端を固定したまま入れ替える（全体の占有幅が変わらない）
- サイズの基準を切り替え
  - オフ：パス本体のサイズ（geometric bounds）
  - オン：線幅・効果を含む見た目のサイズ（visible bounds）
- ライブプレビュー（キャンセルで元の位置に復元）
- 2つの幅が同じときは「中心位置を交換」、異なるときは「両端の位置を保って交換」を初期選択
- 日本語／英語の自動切り替え、各項目にツールチップ

### 処理の流れ

1. 選択が2つの入れ替え可能なオブジェクトかを検証（ロック／非表示は対象外）
2. ダイアログで位置の基準・サイズの基準を選択
3. 元の位置とサイズを控えたうえで移動量を計算し、適用
4. OKで確定、キャンセルで元の位置に復元

### 対象外

- 選択が2つ以外の場合
- 移動できないオブジェクト、bounds を取得できないオブジェクト
- ロック／非表示のオブジェクト、およびその状態の親レイヤー・親グループに含まれるオブジェクト
