### はじめに

Illustrator で作業をしていると、「このラベルとあのラベル、中身を入れ替えたいな」という場面が意外とよくあります。価格表の数値、見出しのキャッチコピー、図版の注釈ラベルなど、**位置はそのままで文字だけを交換したい**ケースです。

手作業だと、片方をコピーしてどこかに退避し、もう片方を貼り付けて、退避したものを元に戻して……と地味に手間がかかります。コピペの途中で書式が変わってしまったり、退避した文字を貼り忘れたりと、ミスも起きがちです。

このスクリプトは、選択した2つのテキストオブジェクトの**中身（文字列）だけ**をワンクリックで入れ替えます。位置・サイズ・書式はそのまま、文字の内容だけがそっくり交換されます。

### 概要：

- 選択中の2つのテキストフレームの文字列を入れ替える
- テキストオブジェクトの位置やサイズ、書式は変更しない
- ダイアログなし、実行するだけで即座に入れ替え

### 使い方：

1. 入れ替えたい2つのテキストオブジェクトを選択する
2. スクリプトを実行する
3. それぞれの文字列が入れ替わる

たったこれだけです。設定項目もダイアログもありません。

### 処理の流れ：

1. ドキュメントが開かれているかをチェック
2. 選択数が2つちょうどであるかをチェック
3. 選択した2つが両方とも `TextFrame` であるかをチェック
4. 双方の `contents`（文字列）を交換

### 注意点：

- 選択するのは **テキストオブジェクトを2つ ちょうど** です。1つだけ、3つ以上、テキスト以外が混ざっている場合はアラートが表示され、処理は行われません。
- 入れ替わるのは `contents`（プレーンな文字列）です。テキストオブジェクト自体の書式（フォント・サイズ・色など）はそれぞれの枠側に残るため、「枠の見た目はそのまま、流し込まれる文字だけが入れ替わる」挙動になります。

### こんな場面で

- ビフォー／アフターの図版で、ラベルだけ取り違えていたとき
- 2つの見出し候補を並べておいて、配置を保ったまま入れ替えて見比べたいとき
- 表組みで隣り合うセルの値を交換したいとき
